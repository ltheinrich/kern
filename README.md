# Kern
General library for Rust

## What is Kern?
Kern is a general-use library for the Rust programming language

## Using Kern
Add to Cargo.toml
```
[dependencies]
kern = "1"
```

## kern::http (feature)
```
[dependencies]
kern = { version = "1", features = ["http"] }
```
